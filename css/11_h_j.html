justify-content :
1)flex-start : 가로로 왼쪽에 정렬
2)flex-end : 가로로 오른쪽에 정렬 (역순)
3)space-between : 왼쪽 오른쪽으로 시작과 끝에 아이템이
붙고 그사이에 아이템들에 공백이 생김
4)space-around : 왼쪽 오른쪽 가운데 아이템전부다 
공백이 생김
5)space-evenly : 가운데 정렬이 되는데 공백을 포함하고
좌우가 좀더 넓은 공백으로 생김
6)center : 가로로 가운데 정렬

채우기 위해 늘어납니다

left : 항목을 왼쪽으로 정렬
center : 항목을 중앙에 정렬
end : 항목을 인라인 방향끝에 정렬
right : 항목을 오른쪽으로 정렬
