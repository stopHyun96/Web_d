- scroll-behavior : 반듯이 쓰시는 속성
- 스크롤바를 실행할때 직선 점프대신에 스크롤위치를
부드럽게 애니메이션 할지를 정합니다
- auto : 스크롤박스내에서 직선점프를 허용
- smooth : 부드러운 애니메이션 효과를 줍니다
(짧은 스크롤보다 긴 스크롤일때 효과적)


- scroll-margin : 스냅위치와 컨테이너 사이의 거리를 지정합니다


	

	

	

	

	

	

	




- scroll-margin-block(start,end) :
스냅위치와 컨테이너 사이의 블록 방향거리를 지정

- scroll-margin-bottom(left,right,top) :
스크롤에 여백하단을 설정
- scroll-margin-inline(start, end) :
스냅위치와 컨테이너 사이의 인라인 방향거리를 지정
- scroll-padding-block(start,end)
:컨테이너에서 하위요소의 스냅위치까지의 거리를 지정
- scroll-snap-align :
- scroll-snap-stop :
- scroll-snap-type : 스크롤을 중지할때 요소가 포커스에 
맞춰지는 방식과 방향을 지정합니다
- x,y : 스크롤 스냅효과를 x,y축에 설정합니다
- block : 스크롤 스냅효과를 블록 방향으로 설정
- inline : 스크롤 스냅효과를 인라인 방향으로 설정
- both : 스크롤 스냅효과를 x,y축모두에 설정합니다
- mandatory : 스크롤 작업이 완료되면 스크롤이 자동으로
스냅지점으로 이동합니다
- proximity : 필수와 유사하지만 엄격하지 않고
스크롤 동작이 끝나면 자동으로 스냅포인트로 스크롤이 이동하지만
스냅포인트 사이에는 스냅효과가 없는 영역이 있습니다
브라우저에 매개변수에 따라 다릅니다
