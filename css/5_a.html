6)animation : 아래에서 표기할 애니메이션 속성들에 속기 속성이다.
6-1)animation-name : 애니메이션 이름
6-2)animation-duration : 애니메이션 지속시간
6-3)animation-timing-function : 애니메시연 속도 곡선
6-4)animation-delay : 애니메이션 지연시간
6-5)animation-iteration-count : 애니메이션 반복 횟수
6-6)animation-direction : 왼쪽부터 시작할지 오른쪽으로 시작할지 결정
6-7)animation-fill-mode : 애니메이션이 재생되지 않을 때 (시작전, 종료후, 또는 둘다) 요소들의 스타일을 정의
6-8)animation-play-state : 애니메이션을 실행할지 중지할지의 상태를 뜻함




7)animation-delay : 애니메이션 지연시간을 설정해서 그 후에 실행할 수 있도록 함
8)animation-direction : 왼쪽부터 시작할지 오른쪽부터 시작할지 결정
8-1)normal : 애니메이션이 정상적으로 재생 (앞으로 (왼쪽에서 오른쪽으로))
8-2)reverse : 역방향으로 재생 (뒤로(오른쪽에서 왼쪽으로))
8-3)alternate : 먼저 왼쪽에서 오른쪽으로 다시 오른쪽에서 왼쪽으로
8-4)alternate-reverse : 오른쪽에서 왼쪽으로 다시 왼쪽에서 오른쪽으로




9)animation-duration : 애니메이션이 실행되는 시간
10)animation-fill-mode : 애니메이션이 재생되지 않을 때
(시작전, 종료후, 또는 둘다) 요소들의 스타일을 정의
10-1)forwards : 마지막에 오른쪽 가서 멈춤 그 상태를 유지한다
(애니메이션 방향 / 반복횟수에 따라 다르다)
10-2)backwards : 첫번재 키프레임 방향을 따라가고 애니메이션 지연할때도 이 값을 유지한다.
10-3)both : 정방향 역방향 모두에 대한 규칙을 따라서 양방향으로
11)animation-iteration-count : 애니메이션을 재생해야하는 횟수를 지정
12)animation-name : 애니메이션 이름
13)animation-play-state : 멈춤과 실행
14)animation-timing-function : 속도를 일정적으로 가느냐 아니면 빠르게가다가 느리게 가느냐...
ease(일정적), ease-in(빠르게 가다가 느리게), ease-out(느리게 가다가 빠르게), ease-in-out(짬뽕(빠르게 가다가 느리게 가다가 빠르게 감));
15)aspect-ratio : 종횡비를 만들어 준다.
width만 설정하고 종횡비를 가지게 만들 때
