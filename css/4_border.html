CSS border

- 테두리 속성을 사용하면 테두리의 스타일 너비 / 색사을 지정할 수 있다.
- 테두리 스타일 종류
1)dotted : 동그란 점 테두리
2)dashed : 점선테두리
3)solid : 단선테두리
4)double : 이중 테두리
5)groove : 3d 그루브 테두리
6)ridge : 3d 능선 테두리
7)inset : 3d 삽입 테두리
8)outset : 3d 아웃셋 테두리
9)none : 테두리업음

- 테두리 스타일 : border-style : solid;
- 테두리 굵기 : border-width : 5px;
- 테두리의 색상 : border-color : red;
- 테두리 분리해서 쓰는법 : 
	border-top-style : 
	border-bottom-style : 
	border-left-style :
