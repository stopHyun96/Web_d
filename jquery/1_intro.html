jquery 튜토리얼

- jquery는 javascript 라이브러리 이다.
- jquery는 javascript프로그래밍을 크게 단순화 한다.
- 아래와 기본 지식이 있어야한다.
(html, css, 자바스크립트)
- 주로 애니메이션적인 요소들에 대해 사용한다.
- 코드가 가볍고 단순하다.(예전에는 플래시라는 프로그램을 사용했지만 코드가 무겁고 swf라는 플레이어가 없으면 실행이 안됨.)
- cdn을 사용해야만 사용할 수 있다.(선언하지 않으면 실행되지 않음)
- cdn사용법은 일반적으로

jquery 구문

- 자바스크립트와 jquery는 형제와 같다.
- 둘의 구분법 : jquery는 $(선택자)가 들어간다.
	(jquery정의 / 엑세스)
예시)
$(this).hide() : 현재 요소를 숨김
$("p").hide() : 모든 p요소를 숨김
$(".class").hide() : 선택자 클래스를 숨김
$("#id").hide() : 선택자 id를 숨김
- 선택기는 이름, id, class, 유형, 속성, 속성값 등을 기반으로 html요소를 찾는데 또는 선택하는데 사용됨.
- 선택기 종류
$("*") : 모든 엘리먼트를 선택
$(this) : 현재 html요소를 선택
$("p.intro") : p중에 intro클래스를 가지는 요소 선택
$("ul li:first") : ul의 첫번째 li요소를 선택
$("tr:even") : 모든 짝수 tr요소를 선택
$("tr:odd") : 모든 홀수 tr요소를 선택
