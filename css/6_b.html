13)break-after : 지정된 요소 다음에 페이지 나누기
단 나누기 또는 영역 나누기를 발생해야 하는 여부를 정한다.
13-1)all : 기본 상자 바로 뒤에 페이지 나누기를 삽입
13-2)always : 항상 요소 뒤에 페이지 나누기를 삽입
13-3)avoid : 요소 뒤의 페이지/열/영역 나누기 방지
13-4)avoid-column : 요소 뒤의 열 나누기 방지
13-5)avoid-page : 요소 뒤의 페이지 나누기 방지
13-6)avoid-region : 요소 뒤의 영역 나누기 방지
13-7)column : 항상 요소뒤에 단 나누기 삽입
13-8)left : 다음 페이지가 왼쪽 페이지로 서식이 지정되도록 요소 뒤에 하나 또는 두개의 페이지 나누기 삽입
13-9)page : 항상 요소 뒹 ㅔ페이지 나누기 삽입
13-10)recto : 다음 페이지가 직사각형 페이지로 서식이 지정되도록 기본 상자 뒤에 하나 또는 두개의 페이지 나누기를 삽입
13-11)right : 다음 페이지가 올바른 페이지로 서식이 지정되도록 요소 뒤에 하나 또는 두개의 페이지 나누기를 삽입
13-12)verso : 다음 페이지가 verso페이지 형식이 되도록 기본상자 뒤에 하나 또는 두개의 페이지 나누기를 삽입
