요소변경

.innerHTML : HTML을 넣어줌
.attribute : html요소의 속성값 변경
.style.property : 스타일 요소의 속성값 변경


요소 추가 및 삭제

.createElement : html요소 생성
.removeChild : html요소 제거
.appendChild : html요소 추가
.replaceChild : html요소 변경
.write(text) : html출력 스트림에 쓰기

addEventListener()

- 이메서드는 지정된 요소에 이벤트 핸들러를 연결한다
- 기존 이벤트 처리기를 덮어쓰지 않고 이벤트 처리기 요소에 연결한다
- 하나의 요소에 여러 이벤트 핸들러를 추가할 수 없다
- 동일한 유형의 많은 이벤트 핸들러 요소, 즉 두개의 클릭 이벤트를 추가할 수 있다.
