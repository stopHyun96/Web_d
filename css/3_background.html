Backgrounds

1)background-color : 요소의 배경색을 지정합니다

color


2)background-image : 요소의 배경으로 사용할 이미지를 지정 기본적으로 설정하지 않으면 이미지는
허락된 가로크기 만큼 반복해서 표현됨.



3)background-repeat : 
repeat-x : 가로반복, repeat-y : 세로반복
no-repeat; 백그라운드 이미지가 한번만 나옴

4)background-attachment : 배경 이미지를 스크롤할지 아니면 고정할지를 지정
fixed(고정), scroll(스크롤바를 올림과 동시에 따라가게 되어있음)

5)background(속성으로 사용)

6)background-position : 백그라운드 이미지에 대한 포지션을 정해줌 가로 세로

7)background-origin : 배경 이미지의 원점위치(배경 위치 영역 지정)를 지정한다.
padding-box : 기본값 배경이미지는 패딩가장자리의 왼쪽 위 모서리에서 시작
border-box : 배경이미지는 테두리의 왼쪽 상단 모서리부터 시작
content-box : 배경이미지는 콘텐츠의 왼쪽 상단 모서리부터 시작


	
padding-box

8)background-clip : 배경이 요소 내에서 확장되어야 하는 범위를 지정
border-box : 기본값 배경이 테두리 뒤로 확장
padding-box : 배경이 테두리의 안쪽 가장자리까지 확장
content-box : 배경이 콘텐츠 상자의 가장자리까지 확장




기본값 배경이 테두리 뒤로 확장







배경이 테두리의 안쪽 가장가리까지 확장







배경이 콘텐츠 상자의 가장가리까지 확장




9)background-size : 배경이미지의 크기를 지정
auto : 자동으로 크기 지정
가로 세로(100% 50%) or (300px 100px)
