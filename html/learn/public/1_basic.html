HTML이란?

- 웹 페이지의 표준 마크업 언어
- Hyper Text Markup Language의 약자
- HTML은 일련의 요소로 구성된다.
- 기본 문법 -
가장 기본적인 문법은 아래와 같습니다.
Contents
- 위 모든 것을 요소(element)라고 한다.
- 시작태그와 종료태그 둘을 합쳐 태그라고 한다.
- 콘텐츠는 내용이다.
- 태그는 속성을 가질 수 있다.

- css

- css는 종속형 스타일 시트(cascading style sheet)의 약자이다.
- 기본적으로 id는 자바스크립트에서 콘트롤하기 위함
- class로 스타일을 입혀주는 경우가 많다.
- sass를 학습할 것 (컴파일 되는 css)
