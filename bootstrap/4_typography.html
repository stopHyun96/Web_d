- .small : 상위태그보다 글자크기를 약 80%정도로 줄어준다.
- .text-left, text-center, text-right, text-*-right
- .text-*-left : 소형, 중형, 대형 또는 초대형 화면의 왼쪽 정렬된 텍스트를 나타나게 한다.
- .text-break : 긴 텍스트가 레이아웃을 깨는 것을 방지한다.
- .text-decoration-none : removes underline
- .text-justify : 긴장문 글이 멀티로 있을 때 왼쪽 오른쪽을 꽉 채워서 써주는 것
- .text-nowrap : 공백을 감싸지 않음
- .text-lowercase : 소문자 
- .text-uppercase : 대문자
- .text-capitalize : 단어에서 앞글자만 대문자
- .text-reset : 텍스트 또는 링크의 색상을 재설정한다.(상위에서 색상 상속)
- .list-inline : 한줄로 보이게끔 설정 ul, ol에 클래스가 들어가고
   li에는 .list-inline-item이 같이 쓰이게 된다.
- .pre-scrollable : 프리태그에 스크롤바를 생성
